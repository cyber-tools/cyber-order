<h2 align="center">
  Install
</h2>

```bash
npm install cyber-worker/work-cli -g
```


<h2 align="center">
  Commands
</h2>

> <h4>cyber命令</h4>

- 封装之后的git脚本:

```bash
cyber git
```
- npm脚本合集:

```bash
cyber npm
```

- doc脚本合集：

```bash
cyber doc
```

  

> <h4>launch命令</h4>

- 拉起当前模块所属的npm页面:

```bash
launch npm
```

- 拉取当前页面所属的git仓库页面:

```bash
launch git
```
